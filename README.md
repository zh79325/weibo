# weibo
weibo
使用方法：
在WeiBoDownload.py，和WeiBoFans.py中搜索以下代码

  weiboinstance=WeiBoInstance('微博账号','微博密码')
  
  然后替换成自己的账号密码在运行该脚本即可
  
  注意：由于微博会更行，多以一些用于解析微博内容的正则表达式可能会过期，所以需要根据实际情况进行调整相关匹配方法
